# Features
 # Authentication

 Sign Up and Sign In functionality using Firebase Authentication.

Users must log in to access the Browse page.

Browse Page

Responsive header with navigation.

Main Movie section with trailer videos embedded in the background.

Play button and More Info button for each movie.

Vertical scrollable movie lists with multiple categories.

Movie suggestions based on user activity.

# Netflix GPT

AI-powered search bar for smart movie recommendations.

Dynamic movie suggestions leveraging GPT integration.

# UI/UX

Responsive and modern interface using Tailwind CSS.

Dynamic components like header, main movie section, secondary containers.

  # Technologies Used

Frontend: React.js, Tailwind CSS

State Management: Redux, Redux Toolkit

Backend / Services: Firebase Authentication, Firebase Hosting

APIs: Movie APIs for fetching movie data and trailers, YouTube embedded videos

Other Tools: Custom React hooks for API integration
